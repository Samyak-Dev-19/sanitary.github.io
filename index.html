FAQ

Quick answers for common mobile layout issues.

Why does my page scroll sideways on mobile?
Usually a fixed-width element (like a wide image, table, or container) is overflowing. Make sure images use max-width: 100% and avoid hard-coded widths.
How do I make my navbar work on phones?
Use a menu button that toggles the nav open/closed. This template includes a small JS toggle and responsive CSS.
Can I deploy this to GitHub Pages?
Yes. Put these files in a GitHub repo and enable GitHub Pages in your repo settings.
